# MSA Demo

Vuejs 를 이용한 사용자 화면, Nodejs 로 개발된 API 서버로 구성 되어 있습니다. Nodejs 는 MongoDB 에 접속하여 데이터를 서비스하는 구조 입니다.

### NODE API Server
사용자 정보를 입력, 수정, 삭제, 조회를 하기 위한 UI

### Vuejs UI Server
사용자 정보에 대한 GET, POST, DELETE 를 제공하는 API 애플리케이션 서비스
